Help

Welcome!

This application is meant to investigate the relation between the behaviour of users on social media and their tendency of spreading malware.

Log in

You need to be authenticated in order to use this service. If you do not have an account, you can create one or log in through your Google Account.

Tweets

You can search the last tweets of a user, based on his or her account name.

Sentiment Analysis

The service provides you with the facility of making sentiment analysis on the tweets identified through the above search.

Github

You can download the source code from a repository on Github, given the account name.

Malware Analysis

The service provides you with the possibility of making a malware analysis of the downloaded source code. Now, is there a relation between the behaviour of the user on the social media and his tendency of spreading malware on Github?
